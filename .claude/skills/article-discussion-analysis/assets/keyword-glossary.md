# Keyword Glossary

DevOps / AI 時代平台文章推薦關鍵字。依文章主題取子集。

| Keyword                      | 中文說明                         | Why It Matters                                           | Learn Next                                  |
| ---------------------------- | -------------------------------- | -------------------------------------------------------- | ------------------------------------------- |
| DevOps                       | 開發與維運的整合實踐             | 現代軟體交付的基礎                                       | 了解其演化方向（Platform Engineering）      |
| Platform Engineering         | 平台工程                         | DevOps 的演化方向，槓桿最高                              | Internal Developer Platform、Golden Path    |
| Internal Developer Platform  | 內部開發者平台（IDP）            | 團隊生產力的核心基礎建設                                 | Backstage、Humanitec、Port                  |
| Golden Path                  | 黃金路徑                         | 讓開發者用對的方式做對的事                               | 實際案例研究                                |
| Paved Road                   | 鋪好的路                         | 降低開發者決策疲勞                                       | Netflix / Spotify 的 paved road 實踐        |
| Infrastructure as Code, IaC  | 基礎設施即程式碼                 | AI 時代最需要被治理的產出                                | Terraform、OpenTofu、Pulumi                 |
| Policy as Code               | 政策即程式碼                     | 合規與治理的自動化                                       | OPA、Conftest、Sentinel                     |
| GitOps                       | 以 Git 為真實來源的維運模式      | 可稽核、可回滾的部署                                     | ArgoCD、Flux                                |
| SRE                          | 網站可靠性工程                   | 可靠性的工程化                                           | SLI/SLO、Error Budget                       |
| SLI / SLO / Error Budget     | 服務指標 / 目標 / 錯誤預算       | 可靠性管理的核心語言                                     | Google SRE Book                             |
| Observability                | 可觀測性                         | 故障排查與 AIOps 的前提                                  | Logs / Metrics / Traces、OpenTelemetry      |
| AIOps                        | AI 驅動的維運                    | 自動化根因分析與告警降噪                                 | Anomaly detection、auto-remediation         |
| Auto-remediation             | 自動修復                         | 降低 MTTR 的關鍵                                         | Runbook automation                          |
| Incident Management          | 事故管理                         | 正式環境品質的最後防線                                   | PagerDuty / Incident.io 實務                |
| RCA / Postmortem             | 根因分析 / 事後檢討              | 組織學習能力的核心                                       | Blameless postmortem                        |
| LLMOps                       | 大型語言模型維運                 | AI 應用上線後的治理                                      | 模型評測、版本管理、prompt 管理             |
| AI Infrastructure            | AI 基礎設施                      | 訓練與推論的底層基礎                                     | GPU cluster、推論服務化                     |
| GPU Scheduling               | GPU 排程                         | 成本與效率的瓶頸                                         | Ray、Kubernetes device plugin               |
| Vector Database              | 向量資料庫                       | RAG 的基礎                                               | pgvector、Pinecone、Weaviate                |
| RAG Pipeline                 | 檢索增強生成管線                 | 企業 AI 應用的主流架構                                   | Chunking、embedding、retrieval 策略         |
| AI SDLC Governance           | AI 軟體開發週期治理              | 差異化的關鍵能力                                         | AI-generated code review、audit             |
| Agent Permission             | 代理人權限控管                   | Agentic 系統的安全邊界                                   | 最小權限、工具白名單                        |
| Context Governance           | 上下文治理                       | 資料不洩漏、提示不被污染                                 | Prompt injection 防禦                       |
| Human-in-the-loop            | 人類監督迴圈                     | AI 系統責任歸屬的機制                                    | 核准流程設計                                |
